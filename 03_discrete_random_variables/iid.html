Independent and Identically Distributed (IID)#

Definition#

Definition 27 (Independent and Identically Distributed (IID))

Let \(X_1, X_2, \ldots, X_n\) be a sequence of random variables.

We say that the random variables are *independent and identically distributed (i.i.d.) if the following two conditions hold:

  1. The random variables are independent of each other. That is, \(P(X_i = x_i | X_j = x_j, j \neq i) = P(X_i = x_i)\) for all \(i, j\).

  2. The random variables have the same distribution. That is, \(\P \lsq X_1 = x \rsq = \P \lsq X_2 = x \rsq = \ldots = \P \lsq X_n = x \rsq\) for all \(x\).

Examples#

Example 1#

Example 6 (Example 1)

The PMF of the height distribution is given by \(\P \lsq X = x \rsq = \frac{1}{10}\) for all \(x \in \{1, \ldots, 10\}\).

Then if you draw 100 people from this population/distribution, with replacement, then this is an example of IID random variables. This is because with replacement, with each draw, the probability will be the same, and the draws are independent of each other.

Confusion in notation is when you index random variables with \(i\). See here sometimes \(X_1, X_2, \ldots, X_{10}\) can mean 10 different people drawn and here.
